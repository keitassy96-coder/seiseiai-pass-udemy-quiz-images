# 生成AIパスポート Udemy 暗記用問題集 解説画像

Udemy講座「生成AIパスポート 暗記用問題集」の解説欄に挿入する画像（q01.png〜q200.png）を配信するためのリポジトリです。

## URL 形式

```
https://keitassy96-coder.github.io/seiseiai-pass-udemy-quiz-images/qNN.png
```

例: `q01.png` → `https://keitassy96-coder.github.io/seiseiai-pass-udemy-quiz-images/q01.png`

## ファイル構成

- `q001.png` 〜 `q200.png` — 各問題に対応する解説図解（600×448 px）
- 4問ずつまとめて生成された 4-in-1 画像を分割したもの
